# %s 칸 수 지정


|  | 칸 수 | 정렬 |
| --- | --- | --- |
| %ns | n칸 확보 | 오른쪽 |
| %-ns | n칸 확보 | 왼쪽 |

# 정수 칸 수 지정


|  | 칸 수 | 정렬 |
| --- | --- | --- |
| %nd | n칸 확보 | 오른쪽 |
| %-nd | n칸 확보 | 왼쪽 |

# 실수 칸 수 지정


|  | 칸 수 | 정렬 |
| --- | --- | --- |
| %nf | n칸 확보 | 오른쪽 |
| %-nf | n칸 확보 | 왼쪽 |
| %.pf | 소수점 p까지 | 오른쪽 |
| %n.pf | n칸 확보,  소수점 p까지 | 오른쪽 |



## 정리

- 칸 수 : n
- 정렬 
- 소수점 :  .n
