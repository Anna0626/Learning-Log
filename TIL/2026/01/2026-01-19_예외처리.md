# 예외란

주로 실행 시에 발생되는 모든 에러 상황을 예외라고 한다. 특정 몇몇 API는 예외를 처리하지 않으면 컴파일 되지 않는 것도 있다. 예외는 객체이다.

## Exception

컴파일러가 예외 처리 코드 여부를 검사하는 예외 

- 클래스의 하위 타입이 바로 프로그래머가 처리할 타입이다. 
- 보통 예외라고 표현할 때 최고 클래스로 Exception 클래스를 말한다. 
- Throwable이라는 클래스는 자식으로 Error가 있기 때문에 최상위 클래스라고 하지 않는다 
## 예외의 종류

- 컴파일러가 관여하는 예외 : Checked Exception
- 컴파일러가 관여하지 않는 예외 : UnChecked Exception

- IOException : 입출력(Input / Output) 작업 중에 발생하는 오류를 나타내는 예외
- RuntimeException : 실행 중 발생하며, 예외 처리가 강제되지 않는(Unchecked) 예외의 최상위 클래스
### 구분 방법

- Exception 하위 클래스 중 RuntimeException이 있다.
- 해당 클래스의 자식 클래스는 모두 알려지지 않은 예외이다.
- 나머지는 모두 알려진 예외이다.


## 예외처리 방법

오류를 막고 다른 부분까지 실행되게 하기 

1. 던지기(throw)
1. 직접 처리 
1. 클래스 만들기 
### 예시

```javascript
package exception;

import java.io.*;

public class Exception04 {
	public static void main(String[] args) {
		MyFile myFile = new MyFile();
		myFile.fileWriter("apple.txt"); // 쓰기
		myFile.fileReader("apple.txt"); // 읽기
	}
}

class MyFile {   
	public void fileWriter(String fileName)  {   //파일 읽기
		FileWriter fw = null;
		try {
			fw = new FileWriter("c:\\Temp\\"+fileName);
			for(int i = 0; i<3; i++) {
				String data = i + "번째 줄을 씁니다. \n";
				fw.write(data);
			}
		} catch (IOException e) {
			System.out.println("저장할 수 없습니다.");
		} finally {
			try {
				fw.close();
			} catch (IOException e) {
				System.out.println("문제가 발생했습니다.");
			}
		}
		System.out.println("저장 완료");
	}
	
	public void fileReader(String fileName) {   //파일 읽기 
		System.out.println("파일을 읽습니다.");
		FileReader fr = null;
		try {
			fr = new FileReader("c:\\Temp\\"+fileName);
			int data = 0;
			while((data = fr.read()) != -1) {      //-1 : 읽을 것이 없을 때 
				System.out.print((char)data); 				
			}
		} catch (FileNotFoundException e) {
			System.out.println("파일을 찾을 수 없습니다. ");
		} catch (IOException e) {
			System.out.println("파일을 읽을 수 없습니다. ");
		} finally {
			try {
				fr.close();
			} catch (IOException e) {
				System.out.println("닫을 수 없습니다. ");
			}
		}
		System.out.println("파일 읽기 완료");
	}
}
```

실행 결과: 

```javascript
저장 완료
파일을 읽습니다.
0번째 줄을 씁니다. 
1번째 줄을 씁니다. 
2번째 줄을 씁니다. 
```
